# bmff of dart

The package provides a library for reading ISO Base Media File Format (BMFF) files.

## Usage

```yaml
dependencies:
  bmff: any
```

```dart
import 'package:bmff/bmff.dart';

// if you want to read a dart.io.File
import 'package:bmff/bmff_io.dart';
```
